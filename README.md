# 📰 News Feed App

A simple and responsive **News Feed Web Application** that fetches and displays the latest news articles using a news API.  
This project provides a clean interface where users can browse trending news, explore categories, and read articles from various sources in real time.

---

# 🚀 Features

- 📰 **Latest News Feed** – Displays the latest news headlines.
- 🔎 **Search Functionality** – Users can search news articles using keywords.
- 📂 **Category Filtering** – Browse news based on categories like Technology, Sports, Business, etc.
- 🌐 **Real-time API Data** – Fetches live news data from an external API.
- 📱 **Responsive UI** – Works across desktops, tablets, and mobile devices.
- 🖼️ **News Cards Layout** – Clean card-based design for article display.
- 🔗 **Direct Article Links** – Open full articles directly from the source.

# 💡 How It Works

1. The application sends a request to the **News API**.
2. The API returns JSON data containing news articles.
3. JavaScript processes the response.
4. News articles are dynamically rendered as **cards** on the webpage.
